# Podcast Setup: Budget-Alternativen (BGM Projekt)

Dieses Setup ist eine preiswerte Alternative zum Rode/Sennheiser-Equipment. Fokus liegt auf einem professionellen XLR-Signalweg für den PC-Arbeitsplatz.

## 1. Audio Interface (Zentrale)
Das Interface wandelt das analoge Mikrofonsignal für den PC um. Es ersetzt das teure Rode AI-Micro.

* **Empfehlung:** [M-Audio M-Track Solo](https://www.thomann.de/de/m_audio_m_track_solo.htm)
* **Preis:** ca. 45 €
* **Warum:** Bietet 48V Phantomspeisung (wichtig für Kondensatormikros) und einen direkten Kopfhöreranschluss ohne Latenz.

## 2. Studio-Mikrofon (Großmembran)
Ersatz für das Rode NT1-A. Perfekt für den klassischen "Radiosound" im Sitzen.

* **Empfehlung:** [the t.bone SC 400](https://www.thomann.de/de/the_tbone_sc400_grossmembranmikro.htm)
* **Preis:** ca. 55 €
* **Warum:** Ein Klassiker für Einsteiger. Die Spinne (Halterung gegen Erschütterungen) ist bereits dabei.
* **Zubehör:** Du benötigst noch ein [XLR-Kabel](https://www.thomann.de/de/the_sssnake_sk233-3_mikrokabel.htm) (ca. 7 €).

## 3. Lavalier-Mikrofon (Ansteckmikro)
Ersatz für das Sennheiser ME 2. Gut für Flexibilität oder wenn man sich beim Reden viel bewegt.

* **Empfehlung:** [Boya BY-M1](https://www.amazon.de/BOYA-BY-M1-Ansteckmikrofon-Kondensator-Smartphone/dp/B006O6Y6M4) (Oft bei Amazon/Fachhandel) oder [the t.bone LC 97 TWS](https://www.thomann.de/de/the_t.bone_lc_97_tws.htm)
* **Preis:** ca. 15–20 €
* **Warum:** Das Boya hat ein extrem langes Kabel und eine eigene Batterie, wodurch es sowohl am Interface als auch direkt am Smartphone/S24 Ultra funktioniert.

---

## ⚙️ Setup-Notizen für Obsidian
* **Verbindung:** Mikrofon -> XLR-Kabel -> Interface -> USB -> PC.
* **Wichtig:** Beim SC 400 am Interface den Schalter für **"48V" (Phantomspeisung)** aktivieren, sonst kommt kein Ton.
* **Gain-Staging:** Den Regler am Interface so einstellen, dass die LED bei normalem Sprechen grün leuchtet und nie rot (Clipping).
