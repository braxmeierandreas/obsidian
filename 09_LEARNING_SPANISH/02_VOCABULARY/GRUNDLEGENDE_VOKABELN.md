# Vokabular: Grundlegende Vokabeln & Phrasen

Tags: #vokabular #a1

Dies ist eine Liste grundlegender spanischer Wörter und Phrasen, die für den Alltag und die erste Kommunikation unerlässlich sind.

---

## Begrüßungen & Verabschiedungen (Saludos y Despedidas)

| Spanisch | Deutsch | Kontext / Anmerkung |
| :--- | :--- | :--- |
| **Hola** | Hallo | Universal, geht immer. |
| **Buenos días** | Guten Morgen | Bis zum Mittagessen. |
| **Buenas tardes** | Guten Tag / Guten Abend | Nach dem Mittagessen bis es dunkel wird. |
| **Buenas noches** | Gute Nacht / Guten Abend | Wenn es dunkel ist, auch zur Begrüßung. |
| **Adiós** | Auf Wiedersehen | Standard-Verabschiedung. |
| **Hasta luego** | Bis später | Sehr gebräuchlich, wenn man sich am selben Tag wiedersieht. |
| **Hasta pronto** | Bis bald | |
| **Chao / Chau** | Tschüss | Informell, sehr verbreitet. |

## Höflichkeit & Wichtige Fragen (Cortesía y Preguntas Importantes)

| Spanisch | Deutsch |
| :--- | :--- |
| **Por favor** | Bitte |
| **Gracias** | Danke |
| **De nada** | Gern geschehen / Keine Ursache |
| **Lo siento / Perdón** | Entschuldigung / Verzeihung |
| **Sí / No** | Ja / Nein |
| **¿Cómo te llamas?** | Wie heißt du? |
| **Me llamo...** | Ich heiße... |
| **¿Cómo estás?** | Wie geht es dir? (informell) |
| **¿Cómo está usted?** | Wie geht es Ihnen? (formell) |
| **Estoy bien, gracias.**| Mir geht es gut, danke. |
| **¿Y tú? / ¿Y usted?** | Und dir? / Und Ihnen? |
| **¿De dónde eres?** | Woher kommst du? |
| **Soy de...** | Ich komme aus... |
| **¿Hablas alemán/inglés?**| Sprichst du Deutsch/Englisch? |
| **No entiendo.** | Ich verstehe nicht. |
| **¿Puedes hablar más despacio, por favor?** | Kannst du bitte langsamer sprechen? |
