## 2. Richtungsangaben (Indicaciones)

| Spanisch | Deutsch |
| :--- | :--- |
| **aquí** | hier |
| **allí / allá** | dort / dorthin |
| **¿Dónde está...?** | Wo ist...? |
| **a la derecha** | nach rechts |
| **a la izquierda** | nach links |
| **todo recto** | geradeaus |
| **cerca de** | in der Nähe von |
| **lejos de** | weit weg von |
| **delante de** | vor |
| **detrás de** | hinter |
| **al lado de** | neben |
| **enfrente de** | gegenüber von |
| **entre** | zwischen |
| **en** | in / an / auf |
| **sobre** | auf (direkt auf etwas) |
| **debajo de** | unter |
| **ir / andar** | gehen |
| **girar** | abbiegen |
| **cruzar** | überqueren |

---

## 3. Nützliche Phrasen

-   `¿Perdón, cómo llego a la estación?` (Entschuldigen Sie, wie komme ich zum Bahnhof?)
-   `Está todo recto y luego a la izquierda.` (Es ist geradeaus und dann links.)
-   `El supermercado está cerca de mi casa.` (Der Supermarkt ist in der Nähe meines Hauses.)
-   `Vivo en un apartamento grande.` (Ich wohne in einer großen Wohnung.)
-   `Voy a la universidad cada día.` (Ich gehe jeden Tag zur Universität.)
