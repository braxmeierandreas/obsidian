# Vokabular: Farben

Tags: #vokabular #a1 #farben

Farben sind wichtige Adjektive, um Dinge zu beschreiben. Wie alle spanischen Adjektive passen sie sich in Geschlecht und Zahl an das Substantiv an, das sie beschreiben.

---

## 1. Grundfarben

| Spanisch (m./f.) | Deutsch | Beispiel |
| :--- | :--- | :--- |
| **rojo/a** | rot | `una manzana roja` (ein roter Apfel) |
| **azul** / azul | blau | `el cielo azul` (der blaue Himmel) |
| **verde** / verde | grün | `la hierba verde` (das grüne Gras) |
| **amarillo/a** | gelb | `un coche amarillo` (ein gelbes Auto) |
| **blanco/a** | weiß | `una casa blanca` (ein weißes Haus) |
| **negro/a** | schwarz | `un gato negro` (eine schwarze Katze) |
| **naranja** / naranja | orange | `una flor naranja` (eine orange Blume) |
| **rosa** / rosa | rosa | `una camisa rosa` (ein rosa Hemd) |
| **morado/a** | lila / violett | `unas uvas moradas` (lila Trauben) |
| **marrón** / marrón | braun | `un oso marrón` (ein brauner Bär) |
| **gris** / gris | grau | `un día gris` (ein grauer Tag) |
| **dorado/a** | golden | `un anillo dorado` (ein goldener Ring) |
| **plateado/a** | silbern | `un reloj plateado` (eine silberne Uhr) |

## 2. Geschlecht und Zahl von Farben

-   **Endung auf -o/-a:** Farben wie `rojo`, `amarillo`, `negro`, `blanco`, `morado`, `dorado`, `plateado` ändern ihre Endung, um sich an das Geschlecht (`-o` für männlich, `-a` für weiblich) und die Zahl (Plural mit `-s`) des Nomens anzupassen.
    -   `el perro negro` (der schwarze Hund)
    -   `la gata negra` (die schwarze Katze)
    -   `los perros negros` (die schwarzen Hunde)
    -   `las gatas negras` (die schwarzen Katzen)

-   **Endung auf Konsonant oder Vokal außer -o/-a:** Farben wie `azul`, `verde`, `naranja`, `rosa`, `marrón`, `gris` ändern ihr Geschlecht nicht. Für den Plural wird einfach ein `-s` angehängt.
    -   `el coche azul` (das blaue Auto)
    -   `la flor azul` (die blaue Blume)
    -   `los coches azules` (die blauen Autos)
    -   `las flores azules` (die blauen Blumen)

---

## 3. Nützliche Phrasen

-   `¿De qué color es...?` - Welche Farbe hat...?
-   `Mi color favorito es el azul.` - Meine Lieblingsfarbe ist Blau.
-   `Quiero una camiseta roja.` - Ich möchte ein rotes T-Shirt.
-   `La pared es blanca.` - Die Wand ist weiß.
