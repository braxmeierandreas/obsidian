# Assignment Checklist

This checklist helps you track your progress for this assignment.

## Planning & Understanding
- [ ] Read assignment brief thoroughly
- [ ] Clarify any unclear points with professor/TA
- [ ] Break down the assignment into smaller tasks
- [ ] Set personal deadlines for each task

## Research
- [ ] Identify key topics for research
- [ ] Gather relevant academic sources (books, articles, journals)
- [ ] Take detailed notes, cite sources
- [ ] Organize research findings

## Drafting
- [ ] Create an outline for your assignment
- [ ] Write the first draft
- [ ] Ensure all parts of the prompt are addressed
- [ ] Check for logical flow and coherence

## Review & Refinement
- [ ] Review against grading rubric
- [ ] Proofread for grammar, spelling, and punctuation errors
- [ ] Check for proper citation and referencing style
- [ ] (Optional) Get feedback from a peer or writing center
- [ ] Make necessary revisions

## Final Steps
- [ ] Format according to guidelines (font, spacing, margins)
- [ ] Convert to required file format (e.g., PDF)
- [ ] Double-check all attachments if submitting online
- [ ] Submit before the deadline
